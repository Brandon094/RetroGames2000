# 🎮 RetroPixel 2000

Catálogo de juegos de PC de la era 2000-2004 con enlaces de descarga a MediaFire.

## ✨ Características

- 🚀 **Lazy Loading** - Carga infinita con Intersection Observer
- 🔍 **Búsqueda en tiempo real** - Encuentra juegos instantáneamente
- 🏷️ **Filtros por género** - RPG, Shooter, Estrategia, Aventura, Simulación
- 📱 **Diseño responsive** - Adaptado a móviles, tablets y escritorio
- 🎨 **Estilo retro-futurista** - Neones, animaciones y efectos glitch
- 💾 **Base de datos JSON** - Fácil de actualizar y mantener
